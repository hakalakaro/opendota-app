OpenDota API

Search Match Information

User can submit ID of the played match of Dota 2 and the application will display information about the searched match, including region, game result, start time of the game, duration of the game and total kills.

The ID of the match can be found when opening a certain match from Open Dota database and looking on the top right where reads "MATCH ID", for example here.
Example ID:s to use: 6942906546, 6928303091, 6922605879.

Search Player Information

User can submit Steam32 ID of a player and the application will display information about the searched player, including their account name, last login, link to profile, rating, amount of played games and total wins/losses.

The player information uses the player's Steam32 ID as input, there are multiple ways online to convert your account profile information into Steam32 ID, for example here.
Example Steam32 ID:s to use: 43635946, 954729093, 993670205.
